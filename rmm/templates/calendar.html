    Календарь

    Этап 0

    Январь - Март 2018

    Odgers Berndtson собирает по открытым источникам сведения о медиаменеджерах, отвечающих критериям Методологии Рейтинга, а также публичную информацию о показателях работы их компаний.

    На этой основе составляется Первичный список участников Рейтинга.

    Этап 1

    Март - Апрель 2018

    Данные о возрасте менеджера, месте его работы и занимаемой позиции подтвержда-ются самими участниками и (в большинстве случаев) руководителями HR-подразделе-ний соответствующих компаний.

    На этом этапе участник может указать еще 2 кандидатов достойных, по его мнению, включения в Рейтинг. При соответствии критериям Методологии Рейтинга организаторы также внесут их в Основной список.

    PwC обеспечит проверку правильности присвоения баллов в соответствии с Методологией.

    Этап 2

    Апрель - Май 2018

    Сформирован Основной список Рейтинга. Для определения фактора «Признание коллег» проводится голосование среди участников Рейтинга.

    PwC предоставляет информацию для расчета множителя рейтинга «Оценка краткосрочных экономических перспектив секторов медиа»; Odgers Berndtson предоставляет информацию для расчета множителя «Оценка востребованности профессиональных компетенций на кадровом рынке». Проводится окончательный расчет рейтинга по всем 5 факторам.

    Аудиторы PwC обеспечат проверку правильности расчетов персональных рейтингов на каждом из этапов в соответствии с разработанной Методологией.
